Seletores CSS

Bem-vindos!

No CSS nós possuimos os sleetores por ID

  • Seletor por Tipo:Busca elementos por uma tag HTML
  • Seletor por ID (#):Busca elemetnos atrvés do atributo "id"
  • Seletor por Classe (.):Busca elementos através do atributo "class"
  • Seletor Universal:Seleciona todos os eleemtnos HTML
  • Seletor de Atributo ([atrib]): Sleleciona elementos que possuem um atributo esepcífico em nosso documento HTML e conseguimos também buscar atributos com um valor específico
